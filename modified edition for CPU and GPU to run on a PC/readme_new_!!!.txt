ПРОЧТИТЕ:
=========

Это обновленная редакция python-файла для запуска чисто на ПК под CPU и/или GPU (без использования всяких Colaboratory и аналогичных сервисов).

В директории по-умолчанию должны находиться вместе python-файл (либо ноутбук), веса обученных моделей и три части архива horse2zebra_*.zip,

которые далее при работе со скриптом автоматически распаковываются в папку horse2zebra_ с нашими изображениями в трейновых и тестовых подпапках.

Данная папка horse2zebra_ со своими подпапками далее используется при обучении, валидации и тестировании нашей модели CycleGAN.

!!! Также для успешного запуска и работы файла с CycleGAN на ПК должна быть установлена необходимая среда и все необходимые библиотеки 
из списка requirements_new.txt

Данные библиотеки можно установить коммандой:

$ conda create --name <env> --file requirements_new.txt

# <env> - имя вашей среды (environment).
# platform: win-64
